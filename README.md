# Aioscrew - Airline Crew Operating System

**Powered by AI from dCortex**

A comprehensive crew management platform for airlines featuring intelligent AI agents for automated claim validation, scheduling optimization, and compliance monitoring.

## Technology Stack

- **Frontend:** React + TypeScript + Vite
- **Backend:** Node.js + Express + AI Agents
- **AI:** dCortex AI Agents powered by Claude Sonnet 4.5
- **Database:** Neon PostgreSQL
- **Orchestration:** LangGraph

## Quick Start

See **SETUP_GUIDE.md** for complete setup instructions.

## Features

- ✅ Real-time AI claim validation (3-5 seconds)
- ✅ 20 specialized AI agents for crew operations
- ✅ Automated compliance and fraud detection
- ✅ Interactive crew scheduling and optimization
- ✅ Multi-role dashboards (Crew, Scheduler, Payroll, etc.)
